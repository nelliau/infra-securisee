# Infra Réseau Sécurisée - B3 CS

## TPs

### [TP1 - Base Setup](./tp1/)
Mise en place de l'infrastructure réseau :
- Topologie GNS3 (routeur, switches, clients)
- Configuration VLANs et routage inter-VLAN
- Mise en place DHCP

### [TP2 - Offensive Sec](./tp2/)
Attaques réseau :
- DHCP Starvation & DHCP Spoofing
- ARP Poisoning & MITM
- DNS Spoofing

### [TP3 - Defensive Sec](./tp3/)
Protections réseau :
- DHCP Snooping
- DAI (Dynamic ARP Inspection)
- IP Source Guard
- STP PortFast & BPDU Guard
- Désactivation CDP
